# KisaanConnect Backend (Enhanced)
## Setup
1. cd backend
2. npm install
3. copy .env.example to .env and set JWT_SECRET and any API keys
4. npm run dev

## Notes
- This is a simple demo backend using a JSON file as storage (db.json). For production, move to MongoDB or PostgreSQL.
